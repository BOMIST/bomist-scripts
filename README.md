## BOMIST Scripts

Utility scripts to be used with [BOMIST](https://bomist.com).

### Examples

#### NodeJs

Install yarn:

```
$ npm i -g yarn
```

```
$ cd api/js
$ yarn
$ yarn ts src/owned_inventory.ts
(or)
$ yarn ts src/boms_importer.ts --csv "<filepath>"
(...or see other scripts)
```

#### Python

```
$ cd api/python
$ python import_parts_capacitors.py
```
